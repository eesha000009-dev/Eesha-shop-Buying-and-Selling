# EeshaMart AI Shopping Assistant - "Eesha"

🛒 **100% FREE & Open Source AI Shopping Assistant**

## Features
- 🔍 Product search from Supabase database
- 🛒 Add to cart with natural language
- 🧠 Powered by TinyLlama AI (runs locally in Space!)
- 🇳🇬 Nigerian context (Naira currency)

## How It Works

Just talk naturally:
- "Show me phones under ₦50,000"
- "Am looking for drones"
- "I need something that flies" → AI understands and searches for drones!
- "Add the first one to my cart"
- "Add both of them"
- "Add all"
- "What's in my cart?"

## 100% FREE
- **NO API keys required**
- Runs TinyLlama-1.1B-Chat directly in this Space
- 100% AI powered natural language understanding
- No pattern matching - real AI comprehension!

## Tech Stack
- **AI Model**: TinyLlama-1.1B-Chat-v1.0 (runs locally)
- **Backend**: FastAPI with Transformers
- **Database**: Supabase
